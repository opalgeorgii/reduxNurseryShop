e-plantShopping is a modern React-based web application for an online plant nursery. Users can browse various categories of plants, add them to a cart, and view total costs. It features a responsive design and integrates Redux for state management.

##Technologies Used
Frontend: React, HTML, CSS
State Management: Redux Toolkit
Styling: CSS, Responsive Media Queries
Package Management: npm
Optional Images: Pixabay, Unsplash
